What is Linux?

Linux is an open-source, Unix-like operating system kernel created by Linus Torvalds and released in 1991. It is the foundation of many operating systems, including Ubuntu, Fedora, CentOS, and Debian. Linux is known for its stability, security, and customization capabilities.

Why Use Linux?

Linux is popular for:

  • Open Source: The source code is available for modification, making it highly customizable and adaptable.
  • Security: Linux is less prone to viruses and malware compared to other operating systems.
  • Stability: Linux is known for its robustness and reliability, especially for server environments.
  • Performance: Linux is lightweight and performs well even on older hardware.
  • Community Support: There is a large community of developers and users offering free support and resources.

Popular Linux Distributions

  • Ubuntu: User-friendly and popular for desktops and servers.
  • CentOS/RHEL: Preferred for enterprise servers and environments.
  • Debian: Known for its stability and used as the base for many other distributions.
  • Fedora: A cutting-edge distribution with the latest software.
  • Arch Linux: A rolling release distribution known for its flexibility and customization.

How to Install Linux

1. Installing Ubuntu

# Download the Ubuntu ISO from the official website
# Create a bootable USB drive using tools like Rufus or Etcher
# Boot the system from the USB drive and follow the installation wizard

2. Installing CentOS

# Download the CentOS ISO from the official website
# Create a bootable USB drive or DVD
# Boot from the USB/DVD and follow the installation instructions

3. Installing Fedora

# Download the Fedora ISO from the official website
# Use a tool like Rufus to create a bootable USB drive
# Boot the system from the USB drive and follow the installation steps

4. Installing Linux on a Virtual Machine

You can use virtualization software such as VirtualBox or VMware to run Linux without affecting your existing operating system. Create a new VM, allocate resources, and mount the Linux ISO to start the installation.

Common Linux Features

  • Package Management: Tools like APT, YUM, and DNF for managing software packages.
  • Shell Environment: Bash and other shells for command-line interaction and scripting.
  • File Permissions: Fine-grained access control over files and directories.
  • Process Management: Commands like ps, top, and htop for monitoring and managing running processes.
  • Customizability: Ability to modify almost every aspect of the system to suit user preferences.

Best Practices

  • Regularly update and patch your system to maintain security.
  • Use strong password policies and enable user authentication.
  • Backup important data frequently and create system snapshots.
  • Learn basic command-line skills to maximize productivity.
  • Keep track of running services and use tools like `top`, `htop`, and `systemctl` for system management.

Essential Linux Commands

File and Directory Management

# List files and directories
$ ls

# List all files including hidden ones
$ ls -a

# Change directory
$ cd 

# Create a new directory
$ mkdir 

# Remove a file
$ rm 

# Remove a directory and its contents
$ rm -r 

# Copy files or directories
$ cp  

# Move or rename files or directories
$ mv  

# Display the current working directory
$ pwd

# Display contents of a file
$ cat 

# View contents of a file with pagination
$ less 

# Search for a string in a file
$ grep "" 

# Display file size and disk usage
$ du -sh

System Information

# Display system information
$ uname -a

# Check memory usage
$ free -h

# Show disk space usage
$ df -h

# Show CPU information
$ lscpu

# Show hardware details
$ lshw

# Check running processes
$ ps aux

# Monitor real-time system processes
$ top

# Display current user and groups
$ whoami

# Show logged in users
$ who

File Permissions and Ownership

# Change file permissions
$ chmod  

# Change file owner
$ chown : 

# Change group ownership
$ chgrp  

# View file permissions and ownership
$ ls -l

Package Management

# Update package list (Debian/Ubuntu)
$ sudo apt update

# Install a package (Debian/Ubuntu)
$ sudo apt install 

# Remove a package (Debian/Ubuntu)
$ sudo apt remove 

# Upgrade all packages (Debian/Ubuntu)
$ sudo apt upgrade

# Update package list (CentOS/RHEL)
$ sudo yum check-update

# Install a package (CentOS/RHEL)
$ sudo yum install 

# Remove a package (CentOS/RHEL)
$ sudo yum remove

Archiving and Compression

# Create a tar archive
$ tar -cvf .tar 

# Extract a tar archive
$ tar -xvf .tar

# Create a compressed tar archive (gzip)
$ tar -czvf .tar.gz 

# Extract a compressed tar archive (gzip)
$ tar -xzvf .tar.gz

# Compress a file using gzip
$ gzip 

# Decompress a gzip file
$ gunzip .gz

# Compress a file using zip
$ zip .zip 

# Extract a zip file
$ unzip .zip

Networking

# Display network configuration
$ ifconfig

# Show active network connections
$ netstat -tuln

# Check connectivity to a host
$ ping 

# Display routing table
$ route -n

# Display current connections and listening ports
$ ss -tuln

# Show public IP address
$ curl ifconfig.me

Process Management

# Kill a process by PID
$ kill 

# Force kill a process by PID
$ kill -9 

# Find a process by name
$ pgrep 

# Stop a process
$ pkill 

# Display process tree
$ pstree

Disk Management

# List disk drives and partitions
$ lsblk

# Show disk usage of all mounted file systems
$ df -h

# Check the disk usage of a specific directory
$ du -sh 

# Mount a disk partition
$ sudo mount  

# Unmount a disk partition
$ sudo umount

System and User Management

# Add a new user
$ sudo adduser 

# Delete a user
$ sudo deluser 

# Change user password
$ sudo passwd 

# Add a user to a group
$ sudo usermod -aG  

# Display users currently logged in
$ who

# Display user session information
$ w

Miscellaneous Commands

# Search for a command or package
$ which 

# Display the manual page for a command
$ man 

# Show a command's usage
$  --help

# Display the last few lines of a log file
$ tail -n 10 

# Display the first few lines of a file
$ head -n 10 

# View command history
$ history
