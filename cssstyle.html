What is CSS?

CSS (Cascading Style Sheets) is the language used to style and design web pages. It controls the layout, colors, fonts, and overall presentation of content.

Why Use CSS?

  • Separates content (HTML) from design.
  • Provides flexibility to create visually appealing designs.
  • Enables responsive layouts for different devices.
  • Supports reusable styles across multiple pages.

Basic CSS Syntax

CSS is written as rules consisting of selectors and declarations:

selector {
    property: value;
}

Types of CSS

  1. Inline CSS: Defined within an HTML tag using the style attribute.
  2. Internal CSS: Written inside a <style> block within the <head> of an HTML document.
  3. External CSS: Stored in a separate file with a .css extension and linked using the <link> tag.

Responsive Design with CSS

Responsive design ensures your website looks great on all devices. Use CSS media queries to apply styles based on screen size:

@media (max-width: 600px) {
    body {
        background-color: lightblue;
    }
}

Common CSS Properties

  • color: Sets the text color.
  • background-color: Sets the background color.
  • font-family: Defines the font style.
  • margin: Creates space around elements.
  • padding: Creates space within elements.
  • border: Adds a border around elements.

Best Practices

  • Keep CSS files modular and organized.
  • Use classes and IDs for targeting specific elements.
  • Leverage CSS variables for consistent theming.
  • Minimize the use of inline styles for maintainability.
