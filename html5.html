What is HTML?

HTML (HyperText Markup Language) is the standard language used to create web pages. It provides the structure for web content, allowing developers to organize text, images, links, and more.

Key Features of HTML

  • Defines the structure of web pages.
  • Supports multimedia elements like images, videos, and audio.
  • Allows linking between pages using hyperlinks.
  • Works seamlessly with CSS and JavaScript to create interactive websites.

Basic HTML Structure

An HTML document starts with a <!DOCTYPE html> declaration and includes the following basic structure:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My HTML Page</title>
</head>
<body>
    <h1>Welcome to My Page</h1>
    <p>This is a sample HTML page.</p>
</body>
</html>

Common HTML Tags

  • <h1> to <h6>: Headings
  • <p>: Paragraph
  • <a href="">: Hyperlink
  • <img src="" alt="">: Image
  • <ul>, <ol>, <li>: Lists
  • <div>: Division/Container

Responsive Design

To make your HTML pages responsive, use the <meta name="viewport" content="width=device-width, initial-scale=1.0"> tag in the <head> section. This ensures your page scales correctly on different devices.

Example: A Simple Responsive Page

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Responsive Example</title>
</head>
<body>
    <header>
        <h1>Responsive Page</h1>
    </header>
    <main>
        <p>This page adjusts to different screen sizes.</p>
    </main>
</body>
</html>

Best Practices

  • Keep your code clean and well-organized.
  • Use semantic HTML tags (<article>, <nav>, <footer>) for better accessibility.
  • Validate your HTML using tools like the W3C Validator.
  • Use comments (<!-- Comment -->) to document your code.
